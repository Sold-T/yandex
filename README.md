**Личная библиотека книг**
Веб-приложение для учета личной коллекции книг с современным REST API и интерфейсом.

**API Endpoints**

GET /api/books/ - список книг

POST /api/books/ - добавить книгу

GET/PUT/DELETE /api/books/{id}/ - работа с конкретной книгой

POST /api/books/{id}/mark_as_read/ - отметить как прочитанную

POST /api/books/{id}/mark_as_unread/ - отметить как непрочитанную
